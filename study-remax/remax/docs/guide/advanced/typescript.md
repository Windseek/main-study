# TypeScript 支持

Remax 默认支持用 TypeScript 开发。在 `create-remax-app` 中我们亦提供了 TypeScript 模板帮助你快速创建项目。
根据你的需要选择对应的小程序模版。

你可以通过 npx 安装：

```bash
$ npx create-remax-app my-app-ts -t
$ cd my-app-ts && npm i
```

使用 yarn 的方式：

```bash
$ yarn create remax-app my-app-ts -t
$ cd my-app-ts && yarn
```
